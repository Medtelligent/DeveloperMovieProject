Developer Test Project

Create a basic ASP.NET/C# Web Forms application for a movie database. The application can be used in two modes - public and admin. The public mode is what a normal user would use to browse/search for movies stored in the database. The admin mode is for an authenticated user who can add/edit/delete movies in the database.

This project is meant to test overall familiarity of the developer using .NET technologies (web forms and MVC) as well as determine the comfort level of the developer with front-end coding through HTML/CSS/Javascript. Any clarifications needed are welcome. Once the project is complete, the result will be reviewed for:

  • Meeting all the requirements as set forth below
  • Code organization, quality and cleanliness
  • Use of any design patterns
  • Database design

Based on the review, a code review session will be conducted with the candidate with any questions and feedback over a screen share or in-person.

Project Requirements

Public Mode

  • List movies categorized by genres on the main index page of the application
  • Movies in each genre should be the 5 recently added movies that genre
  • Click the genre to see all movies in that genre on a new page
  • Ability to search for movies by title. Entering text in search should suggest titles. This should be done through an API to provide the suggestions invoked using AJAX

Admin Mode

  • All admin pages should be secured by Forms login
  • Manage admin users
  • Manage list of genres
  • Ability to add/edit/delete movies
